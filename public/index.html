Empezando

Agrega los archivos sweetbox.min.css y sweetbox.min.js a tu proyecto.

sweetbox.min.css solo agrega estilos al lightbox, así que puedes utilizar los estilos o el framework que desees para tus imágenes.

Estructura HTML

Deberás tener un contenedor principal para las imágenes, el cual debe tener un ID

<section id="my_container"></section>

Dentro del contenedor, tendrás que crear un nuevo elemento que agrupe las imágenes, debe tener la clase .sweetbox-item

Dentro de cada item deberás agregar las dos imágenes (No importa el orden) la imagen pequeña tendrá la clase sweetbox-small-img y la imagen grande sweetbox-large-img
ésta última será la que se mostrará en el lightbox.

Quedaría así
<section id="my_container">
<div class="sweetbox-item">
<img src="...small-image.png" class="sweetbox-small-img">
<img src="...large-image.png" class="sweetbox-large-img">
</div>
<div class="sweetbox-item">
<img src="...small-image.png" class="sweetbox-small-img">
<img src="...large-image.png" class="sweetbox-large-img">
</div>
<div class="sweetbox-item">
<img src="...small-image.png" class="sweetbox-small-img">
<img src="...large-image.png" class="sweetbox-large-img">
</div>

Todos los items que desees...

</section>

Inicializa la librería

var container = document.getElementById('my_container')
sweetbox(container)

Opciones

Sweetbox es un proyecto joven (Versión 1.0.0) por lo tanto no tiene muchas opciones disponibles, se agregarán funcionalidades en el futuro.

Agregar descripción a las imágenes

Html
<div class="sweetbox-item">
<img src="...small-image.png" class="sweetbox-small-img">
<img src="...large-image.png" class="sweetbox-large-img">

<p class="sweetbox-description-img">
Descripción de la imagen
</p>
</div>

Javascript
var container = document.getElementById('my_container')

sweetbox(container {
descriptionImage: true
})
